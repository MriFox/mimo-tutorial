看看MiMo Code能做什么

它不只是聊天，还能帮你写代码、改bug、做项目！

让它写代码

> 帮我写一个Python函数，判断一个数是否是质数

# MiMo Code会回复：
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

让它解释代码

> 解释这段代码：
arr.filter(x => x > 5).map(x => x * 2)

# MiMo Code会回复：
这段代码做了两件事：
1. filter: 筛选出大于5的元素
2. map: 把每个元素乘以2
最终返回一个新数组。

让它改bug

> 我的代码报错：TypeError: Cannot read property 'map' of undefined

# MiMo Code会回复：
问题在于数组可能为空。修复方法：
if (Array.isArray(arr)) {
  return arr.map(...);
}
