10. 语音输入

🎤
打字太累？MiMo Code支持语音输入，动动嘴就能写代码！

2.1 什么是语音输入？

想象一下：你对着电脑说话，MiMo Code就能听懂并执行！

[语音] 帮我写一个Python函数，计算两个数的和
好的！这是计算两数之和的函数...

语音 = 解放双手

开车、吃饭、躺床上都能写代码！

10.2 启用语音输入

只需要一个命令：

/voice

看到语音输入模式启动了吗？现在开始说话吧！

10.3 怎么用？

  1. 输入 /voice 启动语音模式
  2. 对着麦克风说话
  3. 说话停顿后自动发送
  4. MiMo Code会转录并响应
✨
说话要清晰，停顿要明显，这样识别更准确！

10.4 系统要求

语音输入需要一些额外配置：

系统 需要安装
macOS brew install sox
Linux sudo apt install sox
Windows 需要额外配置音频驱动
⚠️ 语音输入需要MiMo登录用户才能使用（MiMo Auto免费用户也可以）

10.5 使用技巧

这些技巧让语音识别更准

掌握这些，识别率提升50%！

  • 清晰发音 - 说标准普通话，别太快
  • 适当停顿 - 句子之间停一下
  • 简洁表达 - 语音适合短指令，复杂内容用键盘
  • 安静环境 - 背景噪音会影响识别
