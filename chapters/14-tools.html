13. 内置工具详解

🔧
MiMo Code自带一堆强大的工具，就像瑞士军刀一样！了解它们，你就能让MiMo做更多事情！

13.1 什么是工具？

工具就是MiMo Code的"超能力"。有了它们，MiMo不仅能聊天，还能：

  • 执行命令
  • 读写文件
  • 搜索代码
  • 浏览网页

工具 = MiMo的手和脚

没有工具，MiMo只能"动嘴"；有了工具，它能"动手干活"！

13.2 工具一览表

工具 作用 使用场景
bash 执行终端命令 运行程序、安装依赖
edit 精确编辑文件 修改代码、修复bug
write 创建/覆盖文件 新建文件、完全重写
read 读取文件内容 查看代码、理解项目
grep 搜索文件内容 查找函数、定位问题
glob 查找文件 按模式找文件
webfetch 获取网页内容 查阅文档、研究资料
question 向你提问 澄清需求、确认选择

13.3 bash工具

最强工具：执行任何终端命令

MiMo可以用它运行npm、git、python等任何命令！

帮我运行npm install安装依赖
好的！正在执行npm install...
[执行命令] npm install
依赖安装完成！

你可以让MiMo执行：

> 运行 npm test
> 执行 python script.py
> 运行 git status
> 安装这个包：npm install lodash

13.4 edit工具

精确修改文件内容，只改你指定的部分：

把第10行的 console.log 改成 console.error
好的！我来精确修改...
[edit工具] 已将第10行的 console.log 改为 console.error
💡 edit工具只替换精确匹配的内容，不会误改其他代码！

13.5 write工具

创建新文件或完全重写文件：

> 创建文件 config.js，内容是数据库配置
> 重写 index.html，使用新的设计风格
⚠️ write工具会覆盖整个文件，小心使用！

13.6 grep工具

在代码库中搜索内容，支持正则表达式：

> 搜索所有包含 TODO 的注释
> 查找哪里用了 fetch 函数
> 搜索所有的 API 接口地址
🔍
grep就像一个超级搜索器，能帮你在千万行代码中快速定位！

13.7 glob工具

按文件名模式查找文件：

> 找到所有的 .js 文件
> 查找 src 目录下的所有 .ts 文件
> 找到所有的测试文件

13.8 实战组合

工具组合使用，效率翻倍！

先用grep找到所有的TODO，然后帮我逐个处理

2. src/api.js:18 - TODO: 实现缓存
...
