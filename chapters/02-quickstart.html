2. 快速上手

🚀
安装好了？太棒！现在让我们和MiMo Code聊聊天，看看这个AI小助手有多聪明！

2.1 启动对话

在终端输入 mimo 启动后，你就可以开始和它对话了。就像和朋友聊天一样简单！

$ mimo
MiMoCode v1.0.0
Type your message and press Enter to start.

> 你好呀！

2.2 试试这些有趣的问题

第一次对话，从这些问题开始

不知道问什么？这些话题能让MiMo Code展示它的能力！

让它做自我介绍

> 你能做什么？给我介绍一下你自己

问个简单的问题

> Python和JavaScript有什么区别？

让它写点代码

> 帮我写一个"Hello World"程序
💡
小技巧：MiMo Code可以写几乎所有编程语言的代码，包括Python、JavaScript、Java、C++等等！

2.3 必会操作

这些操作你必须知道

掌握这几个，你就能流畅使用MiMo Code了！

操作 怎么做 什么时候用
发送消息 输入内容，按回车 任何时候
多行输入 Shift + Enter 输入长代码或长问题时
切换代理 Tab 键 需要不同模式时
退出 /exit 或 Ctrl+C 不想聊了的时候

2.4 实战练习

跟着做一遍，你就入门了！

复制这些对话，看看MiMo Code会怎么回答你

练习1：解释概念

> 用简单的语言解释什么是变量
> 什么是for循环？给我举个例子
> 解释一下什么是函数

练习2：写代码

> 写一个Python程序，输入两个数，输出它们的和
> 用JavaScript写一个倒计时函数
> 帮我写一个HTML页面，有个按钮，点击后显示"你好世界"

练习3：代码解释

> 解释这段代码的作用：
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
💡
试试让MiMo Code解释你在网上看到的代码，这是学编程最快的方式！

2.5 理解MiMo Code的回答

MiMo Code的回答通常很贴心，它会：

  • 用清晰的文字解释概念
  • 给出格式化的代码示例
  • 提供实用的练习建议
  • 回答你的后续问题
🎯
记住：MiMo Code没有"愚蠢的问题"！问得越多，学得越快！
