17. 斜杠命令大全

⚡
斜杠命令是MiMo Code的"快捷操作"，一个命令搞定复杂操作！

17.1 会话管理命令

命令 作用 快捷键
/new 开始新会话 ctrl+x n
/clear 清除当前会话（同/new） -
/sessions 列出所有会话 ctrl+x l
/compact 压缩会话历史 ctrl+x c

17.2 导出分享命令

命令 作用 快捷键
/export 导出对话为Markdown ctrl+x x
/share 分享当前会话 ctrl+x s
/unshare 取消分享 -

17.3 撤销重做命令

后悔药来了！

操作错了？用undo/redo挽回！

命令 作用 快捷键
/undo 撤销上一步操作 ctrl+x u
/redo 重做被撤销的操作 ctrl+x r
💡 undo/redo不仅撤销对话，还会还原文件修改！（需要Git仓库）

17.4 其他实用命令

命令 作用 快捷键
/exit 退出MiMo Code ctrl+x q
/help 显示帮助 ctrl+x h
/models 列出可用模型 ctrl+x m
/themes 切换主题 ctrl+x t
/thinking 显示/隐藏思考过程 -
/init 创建AGENTS.md文件 ctrl+x i
/connect 添加AI提供商 -
/editor 打开外部编辑器 ctrl+x e

17.5 常用组合技

# 保存对话发给同事
/share
/export

# 操作错了想撤销
/undo

# 清空重来
/new

# 换个好看的主题
/themes
