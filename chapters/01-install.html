1.2 安装方法

选择适合你的安装方式

两种方法选一种就行，别纠结！

方法一：一键安装（推荐新手）

打开终端（Windows用PowerShell），复制粘贴这行命令：

curl -fsSL https://mimo.xiaomi.com/install | bash
💡 小贴士：不知道怎么打开终端？Windows按 Win + R，输入 powershell 回车就行！

方法二：npm安装（会编程的同学）

如果你已经装了Node.js，用这个：

npm install -g @mimo-ai/cli
