15. MCP扩展工具

🔌
MCP就像给MiMo Code装"插件"！有了它，MiMo能连接数据库、调用API、使用第三方服务！

15.1 什么是MCP？

MCP（Model Context Protocol）是一个协议，让AI能连接外部工具和服务。

MCP = MiMo的"应用商店"

想让MiMo做什么，就装什么插件！

15.2 常用MCP服务器

MCP 作用 使用场景
Context7 搜索文档 查阅最新文档
Sentry 错误监控 查看线上bug
GitHub 代码仓库 管理PR、Issue

15.3 配置MCP

在配置文件中添加MCP服务器：

{
  "mcp": {
    "context7": {
      "type": "remote",
      "url": "https://mcp.context7.com/mcp"
    }
  }
}
