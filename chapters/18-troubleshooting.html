18. 故障排除

🛠️
遇到问题别慌！这份故障排除指南帮你解决90%的常见问题！

18.1 安装问题

问题1：命令找不到

❌ 错误信息

'mimo' 不是内部或外部命令

✅ 解决方案

1. 重启终端

2. 检查npm全局路径是否在PATH中

3. 重新安装：npm install -g @mimo-ai/cli

18.2 连接问题

问题2：API连接失败

❌ 错误信息

Connection refused / Timeout

✅ 解决方案

1. 检查网络连接

2. 检查API密钥是否正确

3. 尝试使用MiMo Auto（免费通道）

18.3 权限问题

问题3：权限不足

❌ 错误信息

Permission denied / EACCES

✅ 解决方案

1. 使用sudo（Linux/Mac）

2. 修改目录权限

3. 使用管理员权限运行

18.4 工具问题

问题4：文件修改不生效

❌ 现象

MiMo说改了文件，但看不到变化

✅ 解决方案

1. 用 /undo 撤销再重试

2. 检查文件路径是否正确

3. 刷新编辑器

18.5 性能问题

问题5：响应很慢

❌ 现象

MiMo回复很慢，等很久

✅ 解决方案

1. 用 /compact 压缩会话

2. 用 /new 开始新会话

3. 切换到更快的模型

18.6 会话问题

问题6：找不到之前的会话

❌ 现象

关掉终端后找不到之前的对话

✅ 解决方案

1. 用 /sessions 查看所有会话

2. 选择要恢复的会话

3. 重要对话记得用 /export 导出

18.7 终极解决方案

🆘
如果以上方法都无效，试试这个"万能操作"：
  1. 用 /new 开始全新会话
  2. 重新描述你的问题
  3. 如果还是不行，重启终端
  4. 最后手段：重新安装MiMo Code
💡 遇到问题时，把错误信息完整地发给MiMo，它能帮你分析原因！
