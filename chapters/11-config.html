11. 高级配置

⚙️
想让MiMo Code更懂你？配置一下，它就能变成你的专属助手！

11.1 配置文件在哪？

MiMo Code的配置文件藏在这里：

系统 位置
Windows %APPDATA%\mimocode\config.json
macOS/Linux ~/.config/mimocode/config.json

11.2 选择AI模型

模型选择 = 手机套餐

不同套餐有不同能力，选适合你的！

模型 特点 需要什么
MiMo Auto ⭐ 免费、简单、够用 什么都不需要
OpenAI GPT 能力强、贵 API密钥
Claude 擅长长文、贵 API密钥
本地模型 免费、需要显卡 Ollama等

11.3 配置示例

使用OpenAI

{
  "provider": "openai",
  "model": "gpt-4",
  "apiKey": "sk-xxxxx"
}

使用本地模型

{
  "provider": "ollama",
  "baseUrl": "http://localhost:11434",
  "model": "llama2"
}

11.4 项目专属配置

可以在项目根目录创建 .mimocode/config.json：

{
  "memory": { "enabled": true },
  "tasks": { "enabled": true },
  "skills": { "path": "./skills" }
}
📁
这样每个项目可以有不同配置，互不干扰！

11.5 环境变量

也可以通过环境变量配置（适合服务器）：

export MIMO_API_KEY="your-key"
export MIMO_MODEL="gpt-4"
