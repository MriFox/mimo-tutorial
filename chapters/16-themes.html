16. 主题美化

🎨
MiMo Code支持多种主题，让你的终端变得更酷炫！

16.1 内置主题

MiMo Code自带多个主题：

主题名 风格
mimocode 默认主题
system 自动适配终端
tokyonight 深蓝色调
catppuccin 柔和粉彩
gruvbox 复古暖色
nord 北欧冷色
matrix 黑客帝国风

16.2 切换主题

两种方式切换主题

方式1：命令切换

/themes

会弹出主题选择界面，用方向键选择，回车确认。

方式2：配置文件

{
  "theme": "tokyonight"
}

16.3 自定义主题

想要独一无二？可以创建自己的主题！

# 创建主题目录
mkdir -p ~/.config/mimocode/themes

# 创建主题文件
vim ~/.config/mimocode/themes/my-theme.json
✨
推荐新手使用 system 主题，它会自动适配你的终端配色！
