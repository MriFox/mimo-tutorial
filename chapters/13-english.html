13. 使用MiMo Code必备英语

📚
编程世界以英语为主，掌握这些词汇，你就能更顺畅地使用MiMo Code！

13.1 为什么需要学英语？

MiMo Code能理解中文，但编程世界里很多术语都是英文的。学会这些词汇：

  • 看得懂报错信息
  • 能更准确地描述需求
  • 搜索资料时更容易找到答案

别担心！只需要记住最常用的就够了

下面这些词学会，日常使用完全没问题！

13.2 编程基础词汇

英语 发音 中文意思
Variable [ˈveriəbl] 变量
Function [ˈfʌŋkʃn] 函数
Loop [luːp] 循环
Array [əˈreɪ] 数组
String [strɪŋ] 字符串
Integer [ˈɪntɪdʒə] 整数
Boolean [ˈbuːliən] 布尔值（true/false）
Object [ˈɒbdʒɪkt] 对象

13.3 常见报错词汇

看到这些词别慌！知道什么意思就能解决问题

报错词 意思 解决方向
Error 错误 代码有问题，需要修复
Warning 警告 代码能跑但有隐患
Undefined 未定义 变量没有赋值
Null 空值 值是空的
TypeError 类型错误 数据类型不对
Syntax Error 语法错误 代码写法不对
Not Found 找不到 文件或路径不存在
Permission Denied 权限不足 没有操作权限

13.4 和MiMo对话常用句

用这些英文句子，MiMo Code能更准确理解你

请求写代码

Write a function that...     # 写一个函数...
Create a program to...       # 创建一个程序...
Implement a feature for...   # 实现一个功能...

请求解释代码

Explain this code            # 解释这段代码
What does this function do?  # 这个函数做什么？
How does this work?          # 这个怎么工作的？

请求修复bug

Fix this bug                 # 修复这个bug
This code has an error       # 这段代码有错误
Help me debug this           # 帮我调试这个

请求优化代码

Optimize this code           # 优化这段代码
Make it faster               # 让它更快
Improve performance          # 提高性能

13.5 Git常用命令

命令 意思

push 推送（上传到远程）
pull 拉取（下载更新）
branch 分支
merge 合并
clone 克隆（复制仓库）
status 状态

13.6 快速记忆口诀

💡
变量 Variable，函数 Function，循环 Loop 要记牢
数组 Array，字符串 String，对象 Object 不能少
Error 是错误，Warning 是警告，Debug 是调试
Push 推上去，Pull 拉下来，Commit 提交好

13.7 实战练习

试试用英文和MiMo Code对话：

> Explain what is recursion
> Fix this error: Cannot read property of undefined
> Optimize this loop for better performance
