# MiMo Code 新手教程

从零开始学习 MiMo Code AI编程助手的完整教程。

## 在线访问

https://mrifox.github.io/mimo-tutorial/

## 教程内容

| 章节 | 内容 | 预计时间 |
|------|------|----------|
| 1. 安装与配置 | 下载安装MiMo Code，完成基础配置 | 10分钟 |
| 2. 快速上手 | 第一个对话，理解基本工作方式 | 15分钟 |
| 3. 三种代理模式 | Build、Plan、Compose的区别与使用 | 20分钟 |
| 4. 持久化记忆 | 理解MiMo Code如何记住项目和偏好 | 15分钟 |
| 5. 任务管理 | 使用任务系统跟踪项目进度 | 15分钟 |
| 6. 子代理系统 | 创建和管理子代理处理复杂任务 | 20分钟 |
| 7. Compose模式 | 掌握规格驱动开发的工作流程 | 25分钟 |
| 8. 技能系统 | 探索内置技能，创建自定义技能 | 20分钟 |
| 9. 常用命令 | 掌握提升效率的快捷命令 | 10分钟 |
| 10. 语音输入 | 使用语音与MiMo Code对话 | 10分钟 |
| 11. 高级配置 | 自定义配置，连接不同AI模型 | 15分钟 |
| 12. 实战案例 | 通过真实项目巩固所学知识 | 30分钟 |
| 13-17. 进阶内容 | 内置工具、MCP、主题、命令、故障排除 | - |
| 附录：必备英语 | 编程常用词汇和报错信息 | 15分钟 |

## 功能特点

- 17个章节，覆盖MiMo Code核心功能
- 学习进度自动保存（localStorage）
- 5套题库随机抽取的毕业考试
- 毕业证书（90分以上获得）
- 响应式设计，支持手机访问
- 趣味对话框和代码示例

## 技术栈

- 纯HTML/CSS/JavaScript
- localStorage保存学习进度
- GitHub Pages部署

## 本地运行

直接双击 `index.html` 文件即可在浏览器中打开。
